O input é uma tag de entrada que fica dentro do form que pega as informações digitalmente. O name identifica o que tem no formulário. O input é um elemento vazio que pode ter diversos tipos. O correto é colocar um name e um id em cada input pedido.O site ao lado do action abaixo permite que sejam feitos testes. O target é para abrir em outra página.
O fieldset agrupa os campos-faz a linha que envolve como um quadrado.

O text area é uma tag para receber textos maiores...tem alguns atributos que podem ser adicionados....
